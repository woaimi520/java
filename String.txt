String 的比较

==    表示比较引用变量包含的地址是不是相同
equal  表示包含的字符串是不是相等 （String 类覆写object的equal 方法）
String s = new String(“renyu”) 直接在内存的堆上创建一个字符串对象

String a= "renyu" 表示先会在内存的堆区域中的字符串缓冲池中寻找 有就把引用 赋给变量
                   、如果没有才会在堆上去创建对象 浪费内存

s.intern(); 把内存上堆中的对象移动到堆中字符串常量池中，病销毁堆中的对象。这样操作后           可以直接使用“==” 优化新能 因为 equal 方法会拆解字符串 耗时。
